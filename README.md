# mvp-phpcalc

### Expected DOM ids (will gracefully no-op if missing):

 - expression (input/textarea)
 - result (div/span)
 - history-panel (container for history)
- history-list (ul/ol for history items)
 - confirm-overlay (confirmation overlay for clearing history)
 - angle-select (select with values 'deg'|'rad')
 - precision-input (number input)
- format-select (select 'json'|'plain')
 - calc-button (button to evaluate)
 - copy-button (button to copy result)
 - export-button (button to download history)
- import-input (file input for uploading history json)
